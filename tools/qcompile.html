QCompile¶

QCompile - compile .ui and .qrc files to Python

Recursively compiles all Qt .ui and .qrc files in the specified package(s).
Usage:
qcompile <package> … qcompile -h

Example:

$ qcompile .
$ qcompile package1 package2/subpackage
