Status Items¶

Status Items can be used in Widget Rules to control and display status data for LinuxCNC.

Tuples¶

If a status item returns a tuple you can get a single value from the tuple by slicing. For example if the tuple is channel 0 ch[0] you can convert it to a string and get the first entry like this str(ch[0][0]). To get the third item in the tuple you slice at 2 because counting starts at 0 not 1 so str(ch[0][2]) returns the third item in the tuple. For axis items they are in this order X, Y, Z, A, B, C, U, V, W.
